## Array30

與老刀的**官方標準**不兼容的地方如下：

### 修正原表格的兩個錯誤【礴】與【彞】

5-4-3v6-原本是【礡】我「修正」為【礴】，照教育部的規範，【礴】才是正確的。

5^8v2v4-收的是【彝】，我改回【彞】。
  
這是因為古代大無碼（BIG5）收的是異體字。我不管這些歷史遺留問題，都改成正確用字。

### w8②修改成了1️⃣~0️⃣

原本w8②收的是1⑴ 2⑵……；我插入了 12️⃣ 22️⃣

我覺得這比較常用。所以w8②以後的符號都要多按一下🈳。

除此之外，應該都兼容於老刀的**官方標準**了，別按到這些，就和用官方表格的體驗一致。

### no的二級簡碼加了「の」與「№」
```
6v9^
1底 2忻 3の 4產 5□ 6□ 7№ 8庾 9寫 0愎
```

### 補完hg1與w5符號

- hg1增加了✔✖❌⭕⚠️⛔✅❎等符號，寫文件時很好用
- w5增加了⇄☝⬃這些

這些符號都疊在後面，所以應該不影響原表格的順序

### 筆畫字根

如2-2-2-2-i增加了㇅㇍㇈𠃉之類的

### 五碼式符號：

除非是罕用字i，否則行列輸入法不會有5碼字。所以我就把自己常用的emoji塞進5碼字：

- aaaaa是手勢：👍👎💪等
- sssss是表情：😀☺️😄等
- ddddd是電腦符號：💾⚙︎↩⌂等
- fffff是工具符號：✏📝💡🔔等
- ,,,,,是上下標數字：¹²³₁₂₃
- lllll與情緒相關：🩷 💐💩🍌💦
- ppppp是文字：㊙㊟💯，當然也有一些組合用的字元如：幹⃝看⃞屁⃝看⃞
