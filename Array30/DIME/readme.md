## Array（匯入用）.cin

### 加了一堆emoji

aaaaa是手勢：👍👎💪等

sssss是表情：😀☺️😄等

ddddd是電腦符號：💾⚙︎↩⌂等

fffff是工具符號：✏📝💡🔔等

,,,,,是上下標數字：¹²³₁₂₃

lllll與情緒相關：🩷 💐💩🍌💦
